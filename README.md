TruthLens: A Multimodal Neuro-Symbolic Framework for News VeracityTruthLens is an autonomous news verification engine designed to address critical setbacks in modern AI fact-checking, such as LLM hallucinations, fixed-bias sentiment analysis, and visual misdirection. By combining Large Language Models (LLMs), Vision-Language Models (VLMs), and Symbolic logic, TruthLens provides a mathematically grounded veracity score based on factual consensus and forensic visual grounding.

Data Sourcing and Ingestion
The dataset is constructed using a real-time ingestion pipeline rather than a pre-collected static file.
Primary Source: Articles are fetched using the NewsData.io API, which provides access to over 150,000 news sources globally.
Data Scope: The pipeline specifically targets high-velocity news categories—including Politics, War, and Technology—where the risk of misinformation and emotional bias is highest.
Metadata Fields: For every record, the system captures the Title, Full Content, Publication Date, Source Domain, and the Image URL for multimodal forensic analysis.
The dataset is inherently multimodal, requiring a synchronized pair of text and visual data.
Textual Features: Includes average sentence length, neutral-to-subjective word ratios, and extracted atomic claims.
Visual Features: Includes image metadata and the latent representation of the image generated by the Qwen2-VL model.
Dataset Diversity: The testing was conducted on specific high-intensity clusters, such as the "Iran-Israel Conflict" and "Strait of Hormuz" events, to test the system's resilience to high-arousal language.



Key Features
 Neuro-Symbolic Filter: Combines LLaMA-3 claim extraction with spaCy NER verification to eliminate ungrounded "hallucinated" claims.Context-Aware Normalization: Adjusts emotional intensity scores against a dynamically calculated Cluster Baseline to prevent false penalties in crisis reporting.Cross-Modal Grounding: Utilizes Qwen2-VL to perform forensic audits between headlines and images, detecting recycled or mismatched media.Weighted Scoring Engine: A transparent, linear weighted combination ($V = \sum w_i F_i$) prioritizing factual consensus and linguistic neutrality.

Architecture
The pipeline follows a sequential multi-stage process:
Ingestion: Real-time news fetching via NewsData.io.
Clustering: Event-based grouping using SentenceTransformer cosine similarity.
Extraction: Multimodal feature extraction (Text via LLM, Images via VLM).
Verification: Neuro-symbolic filtering of atomic claims.
Scoring: Final veracity calculation and classification (Credible, Questionable, Unreliable).
