TruthLens: A Multimodal Neuro-Symbolic Framework for News VeracityTruthLens is an autonomous news verification engine designed to address critical setbacks in modern AI fact-checking, such as LLM hallucinations, fixed-bias sentiment analysis, and visual misdirection. By combining Large Language Models (LLMs), Vision-Language Models (VLMs), and Symbolic logic, TruthLens provides a mathematically grounded veracity score based on factual consensus and forensic visual grounding.🚀 Key FeaturesNeuro-Symbolic Filter: Combines LLaMA-3 claim extraction with spaCy NER verification to eliminate ungrounded "hallucinated" claims.Context-Aware Normalization: Adjusts emotional intensity scores against a dynamically calculated Cluster Baseline to prevent false penalties in crisis reporting.Cross-Modal Grounding: Utilizes Qwen2-VL to perform forensic audits between headlines and images, detecting recycled or mismatched media.Weighted Scoring Engine: A transparent, linear weighted combination ($V = \sum w_i F_i$) prioritizing factual consensus and linguistic neutrality.🛠️ ArchitectureThe pipeline follows a sequential multi-stage process:Ingestion: Real-time news fetching via NewsData.io.Clustering: Event-based grouping using SentenceTransformer cosine similarity.Extraction: Multimodal feature extraction (Text via LLM, Images via VLM).Verification: Neuro-symbolic filtering of atomic claims.Scoring: Final veracity calculation and classification (Credible, Questionable, Unreliable).
